# Seat Booking System

---

## Overview 
The Seat Booking System is a web-based application built using React and TypeScript that allows employees to book office seats efficiently. It simulates a real-world hybrid workplace where employees can reserve seats based on availability, schedules, and organizational rules.

---

## Problem Statement

With the rise of hybrid work culture, managing office seating manually becomes inefficient and error-prone. Conflicts arise due to:

Limited seating capacity
Different employee schedules
Holidays and leaves

This project solves these challenges by providing a structured and automated seat booking system.

---

## Tech Stack
Frontend: React, TypeScript
State Management: React Context API
Architecture: Component-Based Design

---


## Application Flow
User selects an employee
Selects a week and day
System validates:
Working day
Seat availability
Vacation/holiday status
Displays seat map
User selects a seat
Booking is confirmed and UI updates

---

## Seat Booking Logic
Check if selected seat is available
Validate employee eligibility
Ensure no duplicate bookings
Apply business rules (schedule, holidays, vacation)
Update global state
